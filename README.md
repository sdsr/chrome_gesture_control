# Mouse Gesture Navigation - Chrome Extension

크롬 브라우저에서 마우스 우클릭 제스처를 통해 빠르게 탐색할 수 있는 확장 프로그램입니다.
Naver Whale 브라우저의 마우스 제스처 기능과 유사하게 동작합니다.

## 지원 제스처

| 제스처 | 방향 | 동작 |
|--------|------|------|
| 왼쪽으로 드래그 | <- | 뒤로 가기 |
| 오른쪽으로 드래그 | -> | 앞으로 가기 |
| 아래-오른쪽 | v -> | 탭 닫기 |
| 아래-위 | v ^ | 닫힌 탭 열기 |
| 위로 드래그 | ^ | 새 탭 열기 |
| 위-아래 | ^ v | 새로고침 |
| 오른쪽-위 | -> ^ | 맨 위로 스크롤 |
| 오른쪽-아래 | -> v | 맨 아래로 스크롤 |

## 설치 방법

1. 이 프로젝트 폴더를 다운로드합니다.
2. Chrome 브라우저에서 `chrome://extensions` 페이지로 이동합니다.
3. 우측 상단의 **개발자 모드**를 활성화합니다.
4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.
5. 이 프로젝트 폴더를 선택합니다.

## 사용 방법

1. 아무 웹 페이지에서 **마우스 우클릭을 누른 채로** 원하는 방향으로 드래그합니다.
2. 궤적이 화면에 표시되며, 인식된 제스처 이름이 하단에 나타납니다.
3. 마우스 버튼을 놓으면 해당 동작이 실행됩니다.

## 설정

확장 프로그램 아이콘을 클릭하면 설정 팝업이 열립니다.

- **제스처 활성화/비활성화**
- **궤적 표시 여부**
- **제스처 이름 표시 여부**
- **궤적 색상** 변경
- **궤적 두께** 조절
- **민감도** 조절 (제스처로 인정되는 최소 이동 거리)

## 기술 스택

- Chrome Extension Manifest V3
- Vanilla JavaScript (프레임워크 없음)
- Canvas API (궤적 그리기)
- Chrome APIs: `tabs`, `sessions`, `storage`, `scripting`

## 프로젝트 구조

```
chrome_gesture/
  manifest.json      - 확장 프로그램 매니페스트
  background.js      - 서비스 워커 (탭 조작)
  content.js         - 제스처 인식 (마우스 이벤트 처리)
  content.css        - 궤적 및 오버레이 스타일
  popup.html         - 설정 팝업 HTML
  popup.js           - 설정 팝업 로직
  popup.css          - 설정 팝업 스타일
  icons/             - 확장 프로그램 아이콘
```
